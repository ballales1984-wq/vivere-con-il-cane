## Feature 1: Votazione Anonima

### Modello `PostVote`
- `post`: ForeignKey a BlogPost
- `ip_address`: IP dell'utente (per evitare doppi voti)
- `created_at`: timestamp

### View `vote_post`
- GET `post_id`
- Se IP non ha già votato → crea voto + incrementa contatore
- Ritorna JSON con nuovo conteggio

### Template
- Pulsante "Mi piace" con icona cuore
- Mostra contatore voti

---

## Feature 2: Status Articoli

### Campi aggiuntivi in `BlogPost`
- `status`: choices (draft, published, archived)
- `publish_date`: data di pubblicazione
- `votes_count`: contatore voti (cache per velocità)

### Query automatica
- `BlogPost.published.all()` → solo articoli pubblicati con data ≤ ora

## Feature 3: Pubblicazione Automatica

### Manager personalizzato
- `PublishedManager`: filtra per status='published' E publish_date <= now()
